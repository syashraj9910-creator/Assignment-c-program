# Assignment-04: Student Record Management System

## Objective

To develop a menu-driven Student Record Management System using structures in C. The program performs basic operations such as adding, displaying, searching, updating, and deleting student records.

---

## Problem Statement

Develop a menu-driven application to manage student records using structures. The program should provide the following operations:

- Add Student Record
- Display Student Records
- Search Student by Roll Number
- Update Student Details
- Delete Student Record

---

## Concepts Covered

- Structures
- Arrays
- Functions
- Loops
- Conditional Statements

---

## Algorithm

1. Start the program.
2. Display the main menu.
3. Read the user's choice.
4. Perform the selected operation:
   - Add a student.
   - Display all students.
   - Search a student by roll number.
   - Update student details.
   - Delete a student record.
5. Repeat the menu until the user selects Exit.
6. Stop the program.

---

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Add Student | O(1) |
| Display Students | O(n) |
| Search Student | O(n) |
| Update Student | O(n) |
| Delete Student | O(n) |

---

## Space Complexity

**O(n)**

where **n** is the number of student records.
