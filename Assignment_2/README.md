# Assignment-02: String Utility Toolkit

## Objective

To develop a menu-driven program that performs various string operations such as finding the length of a string, reversing a string, checking for palindrome, counting vowels, digits, spaces, and displaying character frequency.

---

## Problem Statement

Develop a menu-driven application to perform the following string operations:

- Find the length of a string
- Reverse a string
- Check whether the string is a palindrome
- Count vowels
- Count digits
- Count spaces
- Display character frequency

---

## Concepts Covered

- Strings
- Character Arrays
- Loops
- Functions
- Conditional Statements

---

## Algorithm

1. Start the program.
2. Accept a string from the user.
3. Display the menu.
4. Perform the selected operation:
   - Find string length.
   - Reverse the string.
   - Check palindrome.
   - Count vowels.
   - Count digits.
   - Count spaces.
   - Display character frequency.
5. Repeat until the user selects Exit.
6. Stop the program.

---

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Find Length | O(n) |
| Reverse String | O(n) |
| Palindrome Check | O(n) |
| Count Vowels | O(n) |
| Count Digits | O(n) |
| Count Spaces | O(n) |
| Character Frequency | O(n²) |

---

## Space Complexity

**O(n)**

where **n** is the length of the string.

---

## Sample Input

```
Enter String:
Madam 123
```

---

## Sample Output

```
Length = 9

Reverse = 321 madaM

Palindrome = No

Vowels = 2

Digits = 3

Spaces = 1
```
