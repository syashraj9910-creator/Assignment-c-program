# Assignment-10: LeetCode Array Challenge

## Objective

To implement common array operations in C such as finding the largest and smallest elements, reversing an array, removing duplicates, counting element frequency, searching, and sorting.

---

## Problem Statement

Develop a menu-driven program to perform the following operations on an array:

- Display Array
- Find Largest and Second Largest Element
- Find Smallest and Second Smallest Element
- Reverse the Array
- Remove Duplicate Elements
- Count Frequency of Elements
- Search an Element using Linear Search
- Sort the Array in Ascending Order

---

## Concepts Covered

- Arrays
- Functions
- Loops
- Searching Algorithms
- Sorting Algorithms
- Time Complexity Analysis

---

## Algorithm

1. Start the program.
2. Read the number of elements.
3. Store the elements in an array.
4. Display the menu.
5. Perform the selected operation:
   - Display array
   - Find largest and second largest element
   - Find smallest and second smallest element
   - Reverse the array
   - Remove duplicate elements
   - Count frequency of elements
   - Search an element
   - Sort the array
6. Repeat until the user selects Exit.
7. Stop the program.

---

## Time Complexity

| Operation | Time Complexity |
|-----------|-----------------|
| Display Array | O(n) |
| Largest & Second Largest | O(n) |
| Smallest & Second Smallest | O(n) |
| Reverse Array | O(n) |
| Remove Duplicates | O(n²) |
| Frequency Count | O(n²) |
| Linear Search | O(n) |
| Bubble Sort | O(n²) |

---

## Space Complexity

**O(n)**

where **n** is the number of elements in the array.
